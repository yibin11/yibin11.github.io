一.GitHub Pages + Hexo

这种方案的本质是：你在自己电脑上写文章（Markdown格式），用 Hexo 把文章转成漂亮的网页，最后推送到 GitHub 上托管。全程免费，不需要买服务器，不需要买域名（除非你想自定义）。

以下是手把手教学：

第一步：安装必备工具

在开始之前，你的电脑需要安装两个基础环境：

  1. Node.js: 运行 Hexo 的引擎。 官网下载 (选 LTS 版本)。
  2. Git: 用来把你的代码上传到 GitHub。 官网下载。

第五步：进阶

——换个酷炫的主题

Hexo 默认的主题比较普通。你可以去 Hexo Themes 找喜欢的主题。

目前最流行的几个：

  • Butterfly: 功能极其强大，非常华丽。
  • Fluid: 响应式设计，非常有质感。
  • Next: 极简主义，很多技术大牛在用。

换主题的方法：
通常是 git clone 主题的代码到你的 themes 文件夹，然后在 _config.yml 里把 theme: landscape 改成新主题的名字。
