6.导航守卫（全局）

详见：src-6.zip

vue3已经放弃了 next ，直接 return

以前你的写法 (Vue 2) 现在你应该怎么写 (Vue 3) 含义
next() return true (或什么都不 return) 绿灯：正常放行，进入下一个页面。
next(false) return false 红灯：中止跳转，停留在当前页面。
next(‘/login’) return ‘/login’ 拐弯：重定向，强制跳到别的网址。
next({ name: ‘Home’ }) return { name: ‘Home’ } 拐弯：通过名字重定向。

你现在正在开发的模式是：Vue 3 (前端) + Django (后端 API)。

在这种模式下，最标准的登录流程是：

  1. 前端发请求，把用户名密码给后端。
  2. 后端验证成功，返回一个令牌（Token）。
  3. 前端拿到 Token，把它存进 localStorage 里。
  4. 以后每次请求后端接口，前端手动从 localStorage 里把 Token 拿出来，放在请求头（Header）的 Authorization 字段里一起发过去。
